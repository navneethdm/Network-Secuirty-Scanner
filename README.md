# 🛡️ Network Security Scanner

A Python-based tool that scans a local network to identify connected devices, detect open ports, and perform basic risk analysis using Nmap.

---

## 🚀 Features

- Automatic network device discovery
- Port scanning with service detection
- Basic operating system identification
- Risk analysis based on commonly exposed ports
- Structured and readable output

---

## 🛠️ Requirements

- Python 3
- Nmap

Install Nmap:
```bash
sudo apt install nmap
